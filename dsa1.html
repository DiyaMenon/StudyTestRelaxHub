Data Structures And Algorithms

Arrays

An array is a collection of elements stored at contiguous memory locations. It allows multiple values to be stored under a single variable name.

Why Use Arrays?

  • Efficient storage and retrieval of elements.
  • Easy access using indexing.
  • Useful for managing large datasets.

Accessing Elements in an Array (C Program)

#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50}; 
    printf("First element: %d\n", arr[0]); 
    printf("Third element: %d\n", arr[2]); 
    return 0;
}

Explanation

Each element in an array is accessed using its index (starting from 0). The program above prints the first and third elements.

Common Operations on Arrays

  • Insertion
  • Deletion
  • Searching
  • Sorting
  • Traversing

Stacks

A stack is a **Last In, First Out (LIFO)** data structure where elements are added and removed from the same end, called the **top**.

Operations in a Stack

  • Push: Adds an element to the top of the stack.
  • Pop: Removes the top element.
  • Peek: Returns the top element without removing it.
  • Display: Shows all elements in the stack.

Stack Implementation Using Arrays

#include <stdio.h>
#define SIZE 5

int stack[SIZE], top = -1;

void push(int value) {
    if (top == SIZE - 1)
        printf("Stack Overflow\n");
    else
        stack[++top] = value;
}

int pop() {
    if (top == -1)
        printf("Stack Underflow\n");
    else
        return stack[top--];
}

int peek() {
    if (top == -1)
        printf("Stack is Empty\n");
    else
        return stack[top];
}

void display() {
    if (top == -1)
        printf("Stack is Empty\n");
    else {
        for (int i = top; i >= 0; i--)
            printf("%d ", stack[i]);
    }
}

How Stack Functions Work (Array Implementation)

Push: Adds an element at the top by incrementing `top` and storing the value.

Pop: Removes the top element by decrementing `top` and returning the value.

Peek: Returns the element at `top` without removing it.

Display: Prints all elements from `top` to the bottom of the stack.

Stack Implementation Using Linked List

#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node* next;
};

struct Node* top = NULL;

void push(int value) {
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
    newNode->data = value;
    newNode->next = top;
    top = newNode;
}

int pop() {
    if (top == NULL)
        printf("Stack Underflow\n");
    else {
        int value = top->data;
        struct Node* temp = top;
        top = top->next;
        free(temp);
        return value;
    }
}

int peek() {
    if (top == NULL)
        printf("Stack is Empty\n");
    else
        return top->data;
}

void display() {
    struct Node* temp = top;
    while (temp != NULL) {
        printf("%d ", temp->data);
        temp = temp->next;
    }
}

How Stack Functions Work (Linked List Implementation)

Push: Creates a new node, assigns it the value, and sets it as the new `top`.

Pop: Removes the `top` node, stores its value, and updates `top` to the next node.

Peek: Returns the data in the `top` node without modifying the stack.

Display: Traverses the linked list and prints each element.

Real-Life Applications of Stacks

  • Undo/Redo Functionality: Text editors use stacks to track changes.
  • Backtracking: Used in recursive algorithms, such as maze-solving.
  • Function Call Stack: Stores return addresses for function calls.
  • Expression Evaluation: Used in parsing expressions in compilers.
  • Browser History: Stores visited pages in a stack.
