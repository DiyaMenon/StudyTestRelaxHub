Web Programming And Python

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used in web development, data science, artificial intelligence, automation, and more.

Why Use Python?

  • Easy to learn and use.
  • Supports multiple programming paradigms.
  • Large community and vast libraries.
  • Cross-platform compatibility.

How to Use Python?

You can install Python from python.org. To write and run Python code:

  • Use an IDE like **PyCharm, VS Code, Jupyter Notebook**.
  • Run Python scripts in the **terminal** using `python script.py`.
  • Use the **Python interactive shell** by typing `python` in the terminal.

Basic Syntax in Python

Python does not require semicolons and uses indentation to define blocks of code.

Printing in Python

# Print a message
print("Hello, World!")

Variables and Data Types

# Integer, Float, String
num = 10       # Integer
pi = 3.14      # Float
name = "Alice" # String

# Printing variables
print(num, pi, name)

Taking User Input

# Taking input from the user
name = input("Enter your name: ")
print("Hello, " + name + "!")

Conditional Statements (If-Else)

age = int(input("Enter your age: "))

if age >= 18:
    print("You are an adult.")
elif age > 12:
    print("You are a teenager.")
else:
    print("You are a child.")

Loops in Python

For Loop

# Print numbers from 1 to 5
for i in range(1, 6):
    print(i)

While Loop

# Print numbers from 1 to 5 using while loop
i = 1
while i <= 5:
    print(i)
    i += 1

Break and Continue

# Using break
for i in range(1, 10):
    if i == 5:
        break
    print(i)

# Using continue
for i in range(1, 10):
    if i == 5:
        continue
    print(i)

Functions in Python

# Function to add two numbers
def add(a, b):
    return a + b

result = add(5, 3)
print("Sum:", result)

Lists, Tuples, and Dictionaries

Lists (Mutable)

# List Example
fruits = ["Apple", "Banana", "Cherry"]
fruits.append("Mango")  # Adding an element
print(fruits)

Tuples (Immutable)

# Tuple Example
coordinates = (10, 20)
print(coordinates)

Dictionaries (Key-Value Pairs)

# Dictionary Example
student = {
    "name": "Alice",
    "age": 20,
    "course": "CS"
}
print(student["name"])

File Handling

# Writing to a file
file = open("example.txt", "w")
file.write("Hello, Python!")
file.close()

# Reading from a file
file = open("example.txt", "r")
content = file.read()
print(content)
file.close()
