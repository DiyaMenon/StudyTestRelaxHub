If-Else Statement

The if-else statement is used to execute a block of code based on a condition.

Syntax:

            
if (condition) {
    // Code executes if condition is true
} else {
    // Code executes if condition is false
}

Example: Finding the Greatest of Two Numbers

            
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    if (a > b) {
        printf("A is greater");
    } else {
        printf("B is greater");
    }
    return 0;
}

Explanation:

- If `a` is greater than `b`, the first block executes.

- Otherwise, the else block executes.

Else-If Statement

The else-if statement allows multiple conditions to be checked in sequence.

Syntax:

            
if (condition1) {
    // Executes if condition1 is true
} else if (condition2) {
    // Executes if condition2 is true
} else {
    // Executes if all conditions are false
}

Example: Checking a Student's Grade

            
#include <stdio.h>

int main() {
    int marks = 75;

    if (marks >= 90) {
        printf("Grade: A");
    } else if (marks >= 80) {
        printf("Grade: B");
    } else if (marks >= 70) {
        printf("Grade: C");
    } else {
        printf("Grade: F");
    }
    return 0;
}

Why Use Else-If Instead of Multiple If Statements?

Using multiple if statements without else-if can cause unnecessary checks:

            
if (marks >= 90) {
    printf("Grade: A");
}
if (marks >= 80) {
    printf("Grade: B");
}
if (marks >= 70) {
    printf("Grade: C");
}

In this case, if `marks = 95`, all three if statements will execute because each one is checked separately. Using `else-if` ensures that only one block runs.

While Loop

The while loop executes a block of code repeatedly as long as a condition is true.

Example: Print Numbers from 1 to 5

            
#include <stdio.h>

int main() {
    int i = 1;
    while (i <= 5) {
        printf("%d ", i);
        i++;
    }
    return 0;
}

For Loop

The for loop executes a block of code a specific number of times.

Example: Print Numbers from 1 to 5

            
#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    return 0;
}

Switch Case

The switch case statement allows selecting a block of code to execute based on a variable's value.

Example: Displaying Day of the Week

            
#include <stdio.h>

int main() {
    int day = 3;
    switch (day) {
        case 1:
            printf("Monday");
            break;
        case 2:
            printf("Tuesday");
            break;
        case 3:
            printf("Wednesday");
            break;
        default:
            printf("Invalid day");
    }
    return 0;
}

Pattern Example

Pattern:

            
* 
* * 
* * * 
* * * * 
* * * * *

Code:

            
#include <stdio.h>

int main() {
    int i, j;
    for (i = 1; i <= 5; i++) {
        for (j = 1; j <= i; j++) {
            printf("* ");
        }
        printf("\n");
    }
    return 0;
}

Explanation:

- The outer loop (`i`) controls the number of rows.

- The inner loop (`j`) controls the number of stars printed per row.

- After each row, a newline is printed.
