Python Errors

Errors in Python occur when the code is incorrect or behaves unexpectedly. Below are common types of errors:

1. Syntax Errors

# Incorrect Syntax
print("Hello, World!"  # Missing closing parenthesis

Explanation: A syntax error occurs when Python cannot understand the code due to incorrect structure.

2. Runtime Errors

# Division by zero runtime error
a = 10 / 0  # This will cause a ZeroDivisionError

Explanation: This happens during execution. Here, dividing by zero is not allowed.

3. Logical Errors

# Incorrect logic
def multiply(a, b):
    return a + b  # Should be a * b instead of a + b

Explanation: The program runs but gives incorrect output due to wrong logic.

Lists in Python

A list is a collection of items that are ordered and mutable. It can store different data types.

# Creating a list
fruits = ["apple", "banana", "cherry"]

# List functions
fruits.append("orange")  # Adds an item
fruits.insert(1, "grape")  # Inserts at index 1
fruits.remove("banana")  # Removes an item by value
fruits.pop(2)  # Removes item at index 2
fruits.reverse()  # Reverses the list
fruits.sort()  # Sorts the list

print(fruits)

Explanation:

  • append() adds an element to the end of the list.
  • insert() adds an element at a specific index.
  • remove() deletes an element by value.
  • pop() removes an element at a given index.
  • reverse() reverses the list order.
  • sort() sorts the list in ascending order.

Strings in Python

Strings are sequences of characters used to store text. Python provides various string functions.

text = " Hello, Python! "

# String functions
print(text.lower())  # Converts to lowercase
print(text.upper())  # Converts to uppercase
print(text.strip())  # Removes whitespace
print(text.replace("Python", "World"))  # Replaces words
print(text.split(","))  # Splits string into a list
print(len(text))  # Returns length of string

Explanation:

  • lower() converts all letters to lowercase.
  • upper() converts all letters to uppercase.
  • strip() removes whitespace from the beginning and end.
  • replace() replaces a word or character with another.
  • split() splits a string into a list.
  • len() returns the number of characters in a string.

Dictionaries in Python

A dictionary stores key-value pairs and allows fast data retrieval.

# Creating a dictionary
student = {"name": "Alice", "age": 20, "course": "CS"}

# Dictionary functions
print(student.keys())  # Returns all keys
print(student.values())  # Returns all values
print(student.items())  # Returns key-value pairs
student["age"] = 21  # Modifies a value
student.update({"grade": "A"})  # Adds a new key-value pair
del student["course"]  # Deletes a key-value pair

Explanation:

  • keys() returns a list of all keys.
  • values() returns a list of all values.
  • items() returns key-value pairs as tuples.
  • update() adds new key-value pairs or updates existing ones.
  • del removes a key-value pair.

Sets in Python

Sets are unordered collections of unique elements.

# Creating a set
numbers = {1, 2, 3, 4}

# Set operations
numbers.add(5)  # Adds an element
numbers.remove(2)  # Removes an element
print(numbers.union({6, 7}))  # Combines two sets
print(numbers.intersection({3, 4, 5}))  # Finds common elements
numbers.discard(10)  # Removes element if exists, does nothing if not found

Explanation:

  • add() adds a new element to the set.
  • remove() removes an element (raises an error if not found).
  • union() combines two sets without duplicates.
  • intersection() returns common elements between sets.
  • discard() removes an element but does not raise an error if the element is missing.
