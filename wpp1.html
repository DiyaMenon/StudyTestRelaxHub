What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design web pages. It structures web content using elements such as headings, paragraphs, lists, links, images, and tables.

Basic HTML Syntax

HTML uses **tags** enclosed in angle brackets <tag> to define elements. A basic HTML document includes:

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>Heading 1</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Commonly Used HTML Elements

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a href="URL"> - Hyperlink
  • <img src="image.jpg" alt="Description"> - Image
  • <ul> <li> - Unordered list
  • <ol> <li> - Ordered list
  • <table> <tr> <td> - Table
  • <div> - Section container
  • <span> - Inline container

Introduction to CSS

CSS (Cascading Style Sheets) is used to style HTML elements. It controls layout, colors, fonts, spacing, and animations.

Types of CSS

  • Inline CSS: Defined inside the HTML element using the style attribute.
  • Internal CSS: Defined inside the <style> tag within the <head> section.
  • External CSS: Stored in a separate file and linked using <link>.

Example of CSS

/* External CSS file (style.css) */
body {
    background-color: lightgray;
    font-family: Arial, sans-serif;
}
h1 {
    color: darkblue;
    text-align: center;
}

Introduction to JavaScript

JavaScript is a programming language used to add interactivity to websites. It allows dynamic content, form validation, and event handling.

Basic JavaScript Example

This script displays an alert when the page loads:

<script>
    alert("Welcome to Web Development!");
</script>

JavaScript in Action (Button Click)

<button onclick="showMessage()">Click Me</button>

<script>
    function showMessage() {
        alert("Hello, you clicked the button!");
    }
</script>
