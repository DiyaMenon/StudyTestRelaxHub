Introduction to Programming

Overview of C Programming

  • Small and efficient: C is widely used for system and application development.
  • Easy to implement: Simple syntax makes it beginner-friendly.
  • Fast execution: Compiled directly into machine code.
  • Flexible design: Supports both system-defined and user-defined functions.
  • Foundation for other languages: Many modern languages like C++, Java, and Python are influenced by C.

Features of C Language

  • Procedural Language: Follows a step-by-step approach.
  • Portable & Fast: Runs efficiently on different systems.
  • Memory Management: Uses pointers for direct memory access.
  • Rich Library Support: Built-in functions for different tasks.
  • Modularity: Code can be divided into functions for reuse.

Some Terms Are:

Source Code: The human-readable code written by a programmer before compilation.

Output: The result displayed after executing a program.

Function: A block of code designed to perform a specific task.

Library: A collection of predefined functions available in C (e.g., for standard input/output).

Variable: A named storage location used to hold data.

Keyword: Reserved words in C that have a predefined meaning (e.g., int, return).

Functions in C

A function in C is a block of code that performs a specific task. Functions make programs more structured, modular, and reusable.

Why Use Functions?

  • **Code Reusability:** Write once, use multiple times.
  • **Improved Readability:** Makes code easier to understand.
  • **Easier Debugging:** Helps isolate errors in smaller blocks.
  • **Modular Design:** Breaks complex tasks into simpler ones.

Example of a Function in C

            
#include <stdio.h>

// Function to add two numbers
int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 10);
    printf("Sum = %d", result);
    return 0;
}

How Functions Are Created?

A function consists of three parts:

  • **Function Declaration:** Specifies the function name, return type, and parameters.
  • **Function Definition:** Contains the actual code of the function.
  • **Function Call:** Executes the function when needed.

Example: Function with No Return Value

            
#include <stdio.h>

// Function to print a message
void greet() {
    printf("Hello, welcome to C programming!");
}

int main() {
    greet(); // Function call
    return 0;
}

Explanation: The greet() function is defined outside main(). It is called in main() to print a message.
